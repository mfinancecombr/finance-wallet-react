# Finance Wallet Frontend

![Finance Wallet Frontend](finance-wallet-frontend.png)

## Dependencies

- nodejs
- yarn
- [Finance Wallet API][finance-api]
  - via docker compose: `make docker-run`

### Installing dependencies

```bash
make setup
```

## Run it

```bash
make run
```
